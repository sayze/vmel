\subsection{Detailed Description}
File containing a generic error interface which can be consumed by modules. 

\begin{DoxyAuthor}{Author}
Sayed Sadeed \mbox{\hyperlink{struct_error}{Error}} module is simply a store of malloc\textquotesingle{}ed strings. It is upto an individual module to determing how they produce those errors and the nature of them. \mbox{\hyperlink{struct_error}{Error}} simply keeps track of them in a organized structure so that they can be correct freed and accessed. 
\end{DoxyAuthor}
