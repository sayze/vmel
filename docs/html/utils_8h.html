char* file_to_buffer ( const char *  )

Read contents of file to buffer.

This function will read the contents of a passed source file (*.vml). It will simply dump the contents of a file to a buffer. The buffer is created on heap so must be managed by caller. See below example

char *filename = "~/Desktop/run.vml";
char *buffer = file_to_buffer(filename);
free(buffer) // when done.
Parameters
filenamePath to source file.
Returns
Pointer to buffer containing contents of file.

char **  vars,

Replace all the variables in string with corresponding values.

Function is useful for a linear string replace. Note that the placeholder values will be replaced based on position of string in vars.

char **replace = {"sayed", "24"};
char *src = "Hello my name is @0 and I am @1 years old";
char *new = string_map_vars(src, replace, strlen(src), 2);
// new will be equal to
"Hello my name is sayed and I am 24 years old"
Parameters
srcA constant string which contains the variable placeholders.
varsArray of all the string which serve as replacement values.
src_lenLength of src.
vars_lenNumber of string inside vars.
Returns
A malloc'ed char pointer containing the final mapped string. Will return NULL if error occurred.
