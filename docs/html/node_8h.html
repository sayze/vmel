Add an existing Node to the internal NodeMgr store.

This will allow the addition of externally created Node into the NodeMgr. Please note that it is expected *node will be a pointer to Node created on heap. To avoid unexpected behavior it is recommended to use Node_new() then add the returned pointer using this function.
