◆ file_to_buffer()

char* file_to_buffer ( const char *  )

Read contents of file to buffer.

This function will read the contents of a passed source file (*.vml). It will strip out the inline and external comments from the source and save the contents to a buffer. The buffer is created on heap so must be managed by caller. See below example

char *filename = "~/Desktop/run.vml";
char *buffer = file_to_buffer(filename);
free(buffer) // when done.
Parameters
filenamePath to source file.
Returns
Pointer to buffer containing contents of file.
